# magic-cube

## Controls
Esc - Exits game  
Space - Pauses game (currently does nothing)  
L - Left cw turn  
R - Right cw turn  
F - Front cw turn  
B - Back cw turn  
U - Up cw turn  
D - Down cw turn  
P - Reset rubiks cube  
2 - Switch to flat view  
3 - Switch to 3D view  
A - Toggle auto rotation of 3D cube (default off)  
E - Toggle edges (default on)
M - Toggle debugging (default off)  
Mouse actions:  
- Clicking and dragging rotates 3D cube
